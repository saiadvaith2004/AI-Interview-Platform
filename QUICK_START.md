# ⚡ Quick Start Guide

Get up and running in 3 minutes!

## 🚀 Installation

### Step 1: Install Dependencies
```bash
npm install
```

### Step 2: Start the App
```bash
npm run dev
```

### Step 3: Open Browser
The app will automatically open at `http://localhost:3000`

If not, manually navigate to: **http://localhost:3000**

## 🎯 First Interview

1. **Click "Load Sample Resume & Job Description"**
   - This loads demo data so you can test immediately

2. **Click "Start Interview"**
   - You'll see your first question

3. **Type Your Answer**
   - You have 90-180 seconds per question
   - Be specific and provide examples

4. **Submit and Continue**
   - Get instant feedback
   - Answer all 10 questions

5. **View Results**
   - See your readiness score
   - Get actionable feedback

## 📝 Using Your Own Data

Replace the sample data with:
- Your actual resume (copy & paste)
- Target job description (copy & paste)

Then click "Start Interview"!

```bash
npm run preview  # Preview production build
```

## 💡 Tips for Best Results

1. **Use Real Data**: Your actual resume gets better question alignment
2. **Be Thorough**: 50-150 word answers work best
3. **Manage Time**: Aim to finish with 20-30 seconds left
4. **Include Examples**: "For example..." or "In my last project..."
5. **Explain Reasoning**: "Because..." or "This is important because..."

## 🐛 Troubleshooting

**Port 3000 already in use?**
```bash
# Kill the process using port 3000
# Or edit vite.config.js to use a different port
```

**Dependencies won't install?**
```bash
# Try clearing cache
npm cache clean --force
npm install
```

**App not loading?**
- Check console for errors (F12 in browser)
- Ensure Node.js version is 16+
- Try: `rm -rf node_modules && npm install`

## 📊 Scoring System

- **Quality (0-100%)**: Depth + Relevance + Structure
- **Time Efficiency**: On-time = 100%, Overtime = 50%
- **Final Score**: Quality × Base Points × Time Efficiency

### Performance Levels
- **75-100%**: Ready to Hire ✅
- **50-74%**: Needs Development ⚠️
- **<50%**: Not Ready ❌
